question-3:What is event delegation in the DOM?

Event delegation is a technique in the DOM when i use events listens in parent element then it is child elements instead of attaching event listeners to each child individually.

question-4:What is event bubblings in the DOM?

Event bubbling is a concept in the DOM when an event triggered parent elements then a child element propagates("bubbles up")in. The event first runs on the child, then bubbles up to the parent.
